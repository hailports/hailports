# House style — fill this in, save as STYLE.md

The agent reads this to write messages that sound like you, and your connector
can enforce parts of it on the way out. Delete the examples, write your own.

## Voice
- (e.g.) plain, direct, warm. short sentences. contractions are fine.
- (e.g.) no corporate filler, no throat-clearing, no AI/assistant tells.
- Formatting conventions you always want (or never want).

## Names / addressing
- How you refer to specific people (nicknames, first-name-only, etc.).
- Anyone who should only be looped in on specific topics.

## Signature
- The connector appends your signature — the agent should NOT add one.
- Note what the canonical signature is so drafts don't double it up.

## Guardrails
- Anything the agent must never do in a draft (promise timelines, lead an
  external thread, paste raw links, etc.).
